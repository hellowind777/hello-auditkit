# Prompt审计执行清单

> **内容类型**：Prompt（独立提示词、粘贴文本）
> **执行方式**：按ID加载对应注册表，执行所有列出的检查项

## 必须执行的检查项

### 1. 命名检查 (reg-naming.md)

| ID | 检查项 | 条件 |
|----|--------|------|
| N-001 | 命名风格一致 | 总是 |
| N-003 | 无空格特殊字符 | 总是 |
| N-004 | 描述性命名 | 总是 |
| N-005 | 无重复名称 | 总是 |
| N-014 | 标签名规范 | 如有标签 |
| N-015 | 变量名一致 | 如有变量 |

### 2. 引用检查 (reg-reference.md)

| ID | 检查项 | 条件 |
|----|--------|------|
| R-101 | 命名引用存在 | 如有引用 |
| R-102 | 编号引用存在 | 如有编号引用 |
| R-303 | 无循环引用 | 总是 |

### 3. 结构检查 (reg-structure.md)

| ID | 检查项 | 条件 |
|----|--------|------|
| S-201 | 无完全相同规则 | 如有规则 |
| S-305 | 无矛盾 | 总是 |
| S-306 | 无不可能条件 | 总是 |
| S-307 | 无循环逻辑 | 总是 |

### 4. Prompt质量检查 (reg-prompt.md) - 核心

| ID | 检查项 | 条件 |
|----|--------|------|
| P-001 | 详尽约束 | 总是 |
| P-002 | 范围边界 | 总是 |
| P-003 | 禁止捏造 | 如涉及事实内容 |
| P-004 | 歧义处理 | 如有复杂任务 |
| P-005 | 无AI已知内容 | 总是 |
| P-201 | 约束集中 | 总是 |
| P-203 | 禁止语言强度 | 如有关键约束 |
| P-701 | 标签存在 | 如是复杂prompt |
| P-705 | 无未闭合标签 | 如有标签 |
| P-801 | 无填充短语 | 总是 |
| P-802 | 无模糊详尽 | 总是 |
| P-805 | 无矛盾指令 | 总是 |

### 5. 语言检查 (reg-language.md)

| ID | 检查项 | 条件 |
|----|--------|------|
| L-001 | 措辞清晰 | 总是 |
| L-101 | 模糊量词/时间 | 总是 |
| L-201 | 同概念同术语 | 总是 |
| L-303 | 无冗余解释 | 总是 |

### 6. 格式检查 (reg-format.md)

| ID | 检查项 | 条件 |
|----|--------|------|
| F-101 | 标题层级 | 如有标题 |
| F-102 | 表格格式 | 如有表格 |

## 条件执行的检查项

### 如有多阶段/Agentic内容

| ID | 检查项 |
|----|--------|
| P-101 | 停止条件存在 |
| P-102 | 停止条件强度 |
| P-103 | 阶段门条件 |
| P-202 | 约束位置 |
| P-401 | 简短更新 |
| P-403 | 无任务扩展 |
| P-806 | 无分散约束 |
| P-807 | 无弱停止条件 |

### 如有工具使用

| ID | 检查项 |
|----|--------|
| P-301 | 工具优先 |
| P-302 | 工具并行化 |
| P-303 | 写入确认 |

### 如有结构化输出

| ID | 检查项 |
|----|--------|
| P-008 | 输出模式 |
| P-601 | 模式定义 |
| P-602 | 字段区分 |
| P-603 | 空值处理 |

### 如有图表

| ID | 检查项 |
|----|--------|
| F-301 | 节点名匹配文本 |
| F-401 | 所有路径有端点 |
| F-402 | 无无限循环 |

## 执行输出格式

```
Prompt审计执行证据:
- 命名: 6项检查, N个问题
- 引用: 3项检查, M个问题
- 结构: 4项检查, K个问题
- Prompt质量: 12项检查, L个问题
- 语言: 4项检查, J个问题
- 格式: 2项检查, H个问题
[条件检查如适用]
```
