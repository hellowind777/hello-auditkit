# Memory审计执行清单

> **内容类型**：Memory文件（AGENTS.md, CLAUDE.md, GEMINI.md）
> **执行方式**：按ID加载对应注册表，执行所有列出的检查项

## 必须执行的检查项

### 1. 命名检查 (reg-naming.md)

| ID | 检查项 | 条件 |
|----|--------|------|
| N-001 | 命名风格一致 | 总是 |
| N-003 | 无空格特殊字符 | 总是 |
| N-004 | 描述性命名 | 总是 |
| N-005 | 无重复名称 | 总是 |
| N-009 | 无保留字覆盖 | 总是 |
| N-014 | 标签名规范 | 如有标签 |
| N-015 | 变量名一致 | 如有变量 |

### 2. 引用检查 (reg-reference.md)

| ID | 检查项 | 条件 |
|----|--------|------|
| R-101 | 命名引用存在 | 如有引用 |
| R-102 | 编号引用存在 | 如有编号引用 |
| R-104 | 跨文件引用有效 | 如有跨文件引用 |
| R-303 | 无循环引用 | 总是 |

### 3. 结构检查 (reg-structure.md)

| ID | 检查项 | 条件 |
|----|--------|------|
| S-101 | 前置条件优先 | 如有依赖规则 |
| S-104 | 标题层级 | 总是 |
| S-201 | 无完全相同规则 | 如有规则 |
| S-202 | 无语义重复 | 如有规则 |
| S-305 | 无矛盾 | 总是 |
| S-306 | 无不可能条件 | 总是 |
| S-307 | 无循环逻辑 | 总是 |

### 4. Prompt质量检查 (reg-prompt.md) - 核心

| ID | 检查项 | 条件 |
|----|--------|------|
| P-001 | 详尽约束 | 总是 |
| P-002 | 范围边界 | 总是 |
| P-003 | 禁止捏造 | 如涉及事实内容 |
| P-005 | 无AI已知内容 | 总是 |
| P-201 | 约束集中 | 总是 |
| P-203 | 禁止语言强度 | 如有关键约束 |
| P-701 | 标签存在 | 如是复杂内容 |
| P-705 | 无未闭合标签 | 如有标签 |
| P-801 | 无填充短语 | 总是 |
| P-802 | 无模糊详尽 | 总是 |
| P-805 | 无矛盾指令 | 总是 |

### 5. 安全检查 (reg-security.md)

| ID | 检查项 | 条件 |
|----|--------|------|
| X-001 | 无硬编码密钥 | 总是 |
| X-002 | 无暴露凭证 | 总是 |
| X-006 | 无硬编码绝对路径 | 如有路径 |

### 6. 语言检查 (reg-language.md)

| ID | 检查项 | 条件 |
|----|--------|------|
| L-001 | 措辞清晰 | 总是 |
| L-101 | 模糊量词/时间 | 总是 |
| L-201 | 同概念同术语 | 总是 |
| L-303 | 无冗余解释 | 总是 |

### 7. 格式检查 (reg-format.md)

| ID | 检查项 | 条件 |
|----|--------|------|
| F-101 | 标题层级 | 总是 |
| F-102 | 表格格式 | 如有表格 |
| F-205 | 文件编码 | 总是 |

## 条件执行的检查项

### 如有多阶段/Agentic内容

| ID | 检查项 |
|----|--------|
| P-101 | 停止条件存在 |
| P-102 | 停止条件强度 |
| P-202 | 约束位置 |
| P-401 | 简短更新 |
| P-403 | 无任务扩展 |
| P-806 | 无分散约束 |

### 如有工具使用指令

| ID | 检查项 |
|----|--------|
| P-301 | 工具优先 |
| P-303 | 写入确认 |

### 如有命令/路由逻辑

| ID | 检查项 |
|----|--------|
| T-001 | 同概念同术语 |
| T-201 | 无矛盾规则 |
| T-301 | 所有输入有路由 |
| T-302 | 默认路由存在 |

## 执行输出格式

```
Memory审计执行证据:
- 命名: 7项检查, N个问题
- 引用: 4项检查, M个问题
- 结构: 7项检查, K个问题
- Prompt质量: 11项检查, L个问题
- 安全: 3项检查, J个问题
- 语言: 4项检查, H个问题
- 格式: 3项检查, I个问题
[条件检查如适用]
```
