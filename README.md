## Install

Clone/download this repo and put the whole directory into your Codex skills folder:

```text
Windows: %USERPROFILE%\\.codex\\skills\\hello-auditkit
macOS/Linux: ~/.codex/skills/hello-auditkit
```

## Use

- This repository is a Codex Skill directory. Install it as-is.
- Trigger it by mentioning the skill name (e.g. hello-auditkit) or matching its trigger keywords.
