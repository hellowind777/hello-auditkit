# 语言表达检查项注册表

> **唯一权威定义**：所有语言表达检查项在此定义，其他文件只能引用ID。
> **适用范围**：ALL = 所有类型 | S = Skill | P = Plugin | M = Memory

## 检查项列表

### 清晰与精确 (L-0xx)

| ID | 检查项 | 要求 | 严重性 | 适用 |
|----|--------|------|--------|------|
| L-001 | 措辞清晰 | 无歧义短语 | Warning | ALL |
| L-002 | 表达流畅 | 语法正确 | Warning | ALL |
| L-003 | 无拼写错误 | 扫描标识符、标题、关键术语的拼写错误 | Info | ALL |
| L-004 | 无冗余 | 无不必要的重复 | Warning | ALL |
| L-005 | 时态一致 | 全文同一时态 | Info | ALL |

### 歧义检测 (L-1xx)

| ID | 检查项 | 要求 | 严重性 | 适用 |
|----|--------|------|--------|------|
| L-101 | 模糊量词/时间 | 必须具体或有上下文 | Warning | ALL |
| L-102 | 未定义标准 | "appropriate/suitable" 需要定义 | Warning | ALL |
| L-103 | 不完整枚举 | "etc." 应有示例或穷举 | Info | ALL |

### 术语一致性 (L-2xx)

| ID | 检查项 | 要求 | 严重性 | 适用 |
|----|--------|------|--------|------|
| L-201 | 同概念同术语 | 一个概念一个术语 | Warning | ALL |
| L-202 | 已定义领域术语 | 技术术语有解释 | Info | ALL |
| L-203 | 动词形式一致 | 同动作用同动词 | Warning | ALL |
| L-204 | 格式一致 | 同元素用同样式 | Warning | ALL |

### AI执行器意识 (L-3xx)

| ID | 检查项 | 要求 | 严重性 | 适用 |
|----|--------|------|--------|------|
| L-301 | 避免过度规范 | 不指定AI能推断的内容 | Warning | ALL |
| L-302 | 信任AI判断 | 灵活任务用指南而非刚性规则 | Info | ALL |
| L-303 | 无冗余解释 | 不解释AI已知概念 | Warning | ALL |

## 执行方法

```
1. 扫描歧义模式：模糊量词、未定义标准、不完整枚举
2. 检查术语一致性：同概念是否同术语
3. 验证清晰度：是否有无上下文的歧义
4. 应用30%阈值：<30%会误解 → 不是问题
5. 输出：「语言表达: 清晰N问题; 歧义M问题; 术语K问题」
```

## 常见歧义模式

| 模式 | 示例 | 问题 |
|------|------|------|
| 模糊量词 | "一些"、"很多"、"适当" | 无具体数量 |
| 模糊时间 | "很快"、"稍后"、"最近" | 无具体时间 |
| 未定义标准 | "高质量"、"好的代码" | 无评判标准 |
| 不完整列表 | "A、B等" | 边界不清 |

## 不应标记的情况

| 模式 | 原因 |
|------|------|
| AI能从上下文推断 | AI能力 |
| <30%会误解 | 低于阈值 |
| 有示例隐含的标准 | 示范性示例 |
| 设计选择 | 有意为之 |
